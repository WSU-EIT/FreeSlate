Status & badges

In Slate

Drive checklist rows + statuses from a query of the applicant's materials/checklist, output with Liquid into this markup.

Map each Slate status value to a badge class (--done, --wait, --req, --na, --new).

Decision cards: use population rules so admit/deny/waitlist each get the right card. Never show the admit card to a denied applicant.
