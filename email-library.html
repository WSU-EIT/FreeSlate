Slate Editor Hints

Deliver email blocks

In Deliver

Stack blocks inside one 600px wrapper table; paste into the Deliver message HTML source.

Inline CSS only. No <style> blocks, no flexbox/grid — tables for layout.

Merge fields like {{first}} drop straight into the copy.

Add role="presentation" to every layout table (done here) so screen readers skip the grid.

For full Outlook bulletproofing add MSO conditional comments around buttons; test with Litmus before a real send.
