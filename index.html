Slate Editor Hints

Kit overview

How the kit works

The kit is three files linked once in build.xslt: wsu-forms.css (forms), wsu-chrome.css (header/footer), and the XSLT itself. Once deployed, every form is styled automatically.

Most native Slate fields need no custom HTML — the Slate mapping layer styles them. Use the snippets below only for things Slate can’t build natively (cards, banners, notices, columns).

Paste snippets into a form’s Edit HTML block, an Instructions field set to HTML, or a Custom Content area.

Starter snippet — a labeled field

</div>
