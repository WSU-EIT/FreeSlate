Can you do this in Slate?

Yes. Slate-hosted pages render whatever HTML/CSS/JS you give them. The catch is where the pieces go — open the Slate hints tab for the exact steps and copy-paste markup. The short version:

  • Markup → paste into an Instructions field switched to HTML, a Custom Content block, or a portal page body.
  • CSS → already global (it's in the kit), so the gallery/carousel classes just work.
  • JS (carousel only) → add wsu-carousel.js to the form's Edit Scripts, or a /path/default.js. It auto-initializes every .wsuf-carousel on the page.
  • Images → upload to Slate Files and reference by URL, or use a Slate image/widget field. Always include alt text.
⚠
Accessibility is the whole game with carousels

Auto-advancing carousels that can't be paused fail WCAG 2.2.2. This one is manual by default, the Play toggle is pausable, it stops on hover/focus, and it respects reduced-motion. Keep those four behaviors if you customize it.

Slate Editor Hints

Gallery & carousel

Where each piece goes

HTML: Instructions field (toggle to HTML / source view), a Custom Content block, or a portal page body.

CSS: already loaded globally by the kit — the wsuf-gallery / wsuf-carousel classes work with no extra upload.

JS (carousel only): add wsu-carousel.js to the form\u2019s Edit Scripts/Styles, or a /path/default.js. It auto-inits every .wsuf-carousel.

Images: upload to Slate Files, reference by URL. Always add descriptive alt; decorative ones get alt="".

Accessibility: never auto-advance without a pause control (WCAG 2.2.2). This carousel is manual by default and reduced-motion aware.

<div class="wsuf-gallery">

  <a href="big-2.jpg"><img src="thumb-2.jpg" alt="Martin Stadium on game day"></a>

Carousel (needs wsu-carousel.js)

<div class="wsuf-carousel" role="group"
     aria-roledescription="carousel" aria-label="Campus photos">
  <div class="wsuf-carousel__viewport" aria-live="polite">
    <div class="wsuf-carousel__track">
      <div class="wsuf-carousel__slide">
        <img src="1.jpg" alt="Pullman campus aerial">
        <p class="wsuf-carousel__caption">Pullman campus.</p>
      </div>
      <div class="wsuf-carousel__slide">
        <img src="2.jpg" alt="Glenn Terrell Mall">
        <p class="wsuf-carousel__caption">The heart of campus.</p>
      </div>
    </div>
  </div>
  <button type="button" class="wsuf-carousel__btn wsuf-carousel__btn--prev" aria-label="Previous slide">‹</button>
  <button type="button" class="wsuf-carousel__btn wsuf-carousel__btn--next" aria-label="Next slide">›</button>
  <div class="wsuf-carousel__controls">
    <button type="button" class="wsuf-carousel__play" aria-pressed="false"><span aria-hidden="true">▶</span> <span class="wsuf-carousel__play-label">Play</span></button>
    <div class="wsuf-carousel__dots" aria-label="Choose slide"></div>
  </div>
</div>
