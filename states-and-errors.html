Validation

Error summary

When a form fails validation on submit, a focused, linkable summary appears above the form. Each item links directly to the offending field. Screen readers hear it immediately (role="alert").

Error summary + inline errors ✓ role=alert, focus target

There's a problem — 3 fields need attention

  • First name — this field is required.
  • Email address — enter a valid email address.
  • ZIP code — enter a 5-digit ZIP code.
